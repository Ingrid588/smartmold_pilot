## 📋 更新摘要（2026-01-07）

本次更新按照 **MIL Scientific Injection Molding Standard** 完善了数据模板，新增关键元数据和辅助指标。

---

## ✅ 新增字段明细

### 1. Step 1: 粘度曲线 (Viscosity Curve)

**新增列：切换位置 (Switch Position)**

| 字段名 | 单位 | 说明 |
|--------|------|------|
| 速度 (%) | % | 射速百分比 |
| 实际速度 (mm/s) | mm/s | 实际射速 |
| **切换位置 (mm)** | **mm** | **V/P切换位置** ← 新增 |
| 填充时间 (s) | s | 填充时间 |
| 峰值压力 (Bar) | Bar | 注塑峰值压力 |

**用途：** 确保每次射胶的V/P切换位置一致，用于校验测试稳定性。

---

### 2. Step 4: 工艺窗口 (Process Window)

**新增列：保压时间 (Hold Time)**

| 字段名 | 单位 | 说明 |
|--------|------|------|
| 射速 (mm/s) | mm/s | 注塑速度 |
| 保压压力 (Bar) | Bar | 保压压力 |
| **保压时间 (s)** | **s** | **保压持续时间** ← 新增 |
| 产品质量 | - | Pass/Fail |

**用途：** 压力窗口必须记录保压时间，否则压力参数没有参考意义。

---

### 3. Step 7: 锁模力 (Clamping Force)

**新增列：产品重量 (Part Weight)**

| 字段名 | 单位 | 说明 |
|--------|------|------|
| 锁模力 (吨) | 吨 | 锁模力 |
| **产品重量 (g)** | **g** | **实际称重** ← 新增 |
| 飞边情况 | - | Yes/No |

**用途：** 重量变化往往早于肉眼可见的飞边，通过重量辅助判断临界点。

---

## 💡 使用提示

### Excel 上传流程
1. 打开 `SmartMold_数据模板_v4.xlsx`
2. 填写"项目综合信息"（产品、模具、材料、机台）
3. 按顺序填写 Step 1-7 测试数据
4. 在 SmartMold 应用中点击"上传Excel"
5. 系统自动计算派生指标（粘度、剪切率等）
6. 生成专业MIL验证报告（PDF）

### 关键注意事项
- **切换位置**：每次测试应保持一致
- **保压时间**：记录实际保压持续时间
- **产品重量**：精确到0.01g
- **模穴数**：格式如"4"或"1+1"
